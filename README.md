# ED2S - Site Vitrine

Site web vitrine moderne pour ED2S (étanchéité toitures, terrasses, bâtiments en Savoie).

## Démarrage

1. Installez les dépendances :
   ```bash
   npm install
   ```
2. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```

## Tech Stack
- Next.js
- React
- Tailwind CSS
- TypeScript

## Charte graphique
- Orange vif : #FFA100
- Gris neutre : #747678
- Police : Montserrat, Open Sans, Lato
